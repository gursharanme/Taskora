# 📝 Taskora – Task Management Web App

Taskora is a responsive and interactive task management web application built using **HTML, Tailwind CSS, and JavaScript**. It enables users to efficiently create, manage, and track daily tasks through a clean and intuitive interface.

🔗 **Live Demo:** [https://taskora-51k.pages.dev/](https://taskora-51k.pages.dev/)

---

## 🚀 Overview

Taskora is designed to demonstrate practical frontend development concepts including DOM manipulation, client-side state management, and responsive UI implementation.

The application uses the browser’s **localStorage API** to persist data, ensuring tasks remain saved even after refreshing or closing the browser.

This project focuses on:

* Clean UI structure
* Responsive design across devices
* Efficient event handling
* Maintainable JavaScript logic

---

## ✨ Features

* ➕ Add new tasks
* ❌ Delete tasks
* ✅ Mark tasks as complete/incomplete
* 💾 Persistent task storage using localStorage
* 📱 Fully responsive design (Mobile, Tablet, Desktop)
* 🎨 Clean UI built with Tailwind CSS

---
