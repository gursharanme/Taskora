# 🚀 Project Overview

**Taskora** is a simple yet powerful task management web application designed to help users organize and track their daily tasks efficiently.

The application demonstrates core **frontend development principles** including DOM manipulation, event handling, responsive UI design, and client-side data persistence.

Tasks are stored using the browser’s **localStorage API**, ensuring that user data remains saved even after refreshing or reopening the browser.

The goal of this project is to showcase practical **JavaScript functionality combined with modern UI design**.

---

# ✨ Features

* ➕ Add new tasks quickly
* ❌ Delete tasks when completed or no longer needed
* ✅ Mark tasks as complete or incomplete
* 💾 Persistent storage using **localStorage**
* 📱 Fully responsive design across all devices
* 🎨 Clean and minimal UI built with **Tailwind CSS**
* ⚡ Fast and lightweight performance

# 💾 Data Persistence Workflow

```

User adds a task
↓
JavaScript updates task list
↓
Tasks are saved in localStorage
↓
Tasks remain stored after page refresh

````

This ensures a smooth user experience without requiring a backend database.


## 2. Open the project

Since this is a static frontend project, you can simply open:


